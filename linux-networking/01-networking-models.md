# Networking Models

## Models

```
+-------+--------------+-------------+-----------------------+----------------------------+
| Layer |     OSI      |   TCP/IP    | Delivery              | Standard Protocols         |
+-------+--------------+-------------+-----------------------+----------------------------+
| L7    | Application  | Application | Process to Process    | HTTP, ICMP, WhatsApp, etc. |
+-------+--------------+-------------+-----------------------+----------------------------+
| L6    | Presentation |             |                       |                            |
+-------+--------------+-------------+-----------------------+----------------------------+
| L5    | Session      |             |                       |                            |
+-------+--------------+-------------+-----------------------+----------------------------+
| L4    | Transport    | Transport   |                       | UDP / TCP                  |
+-------+--------------+-------------+-----------------------+----------------------------+
| L3    | Network      | Network     | Host to Host (N hops) | IP / IPv6                  |
+-------+--------------+-------------+-----------------------+----------------------------+
| L2    | Data         | Data        | Node to Node (1 hop)  | Ethernet                   |
+-------+--------------+-------------+-----------------------+----------------------------+
| L1    | Physical     | Physical    |                       |                            |
+-------+--------------+-------------+-----------------------+----------------------------+
```

> No layer is aware of the layers above of below it.
